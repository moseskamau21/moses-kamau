Project Overview: B-FIT GYM Manager

B-FIT GYM Manager is a mobile-first gym management system designed specifically for Kenyan fitness centers. The application automates registration, integrates M-Pesa payments, implements QR code attendance tracking, and provides comprehensive admin dashboards for business analytics.

Key Innovation: Contextually appropriate technology - affordable, mobile-optimized, and designed for local payment methods and connectivity constraints.

Tech Stack: React.js (Frontend), Node.js & Express (Backend), JSON Storage (Database), M-Pesa Daraja API (Payments)
